# AWS Basics

## EC2 Setup (Basic Understanding)
- Amazon EC2 is used to create virtual servers in the cloud
- We can launch an instance by selecting OS, instance type, and key pair
- Security groups are used to control access (like firewall rules)
- Instances can be started, stopped, and accessed remotely

## S3 Usage
- Amazon S3 is used for cloud storage
- We can create buckets to store files like images, documents, etc.
- Files can be uploaded, downloaded, and managed easily
- S3 provides secure and scalable storage

## Learning Outcome
- Understood basic cloud computing concepts
- Learned how AWS services like EC2 and S3 are used
- Gained foundational knowledge of hosting and storage in cloud
